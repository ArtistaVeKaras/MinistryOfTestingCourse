# Tech Debt Basics: A Beginner's Guide

Learn the basics of tech debt, HTTP requests, databases, APIs, and browser storage with these resources. This guide will help you understand the foundational concepts needed to manage technical debt effectively.

## ✅ **Step 1: Learn About HTTP Requests and Responses**

- 📘 [**Understanding HTTP: Basics for Beginners** – DEV Community](https://dev.to/devlawrence/understanding-http-basics-for-beginners-3f28): A friendly introduction to how HTTP works, including requests, responses, and status codes [1](https://dev.to/devlawrence/understanding-http-basics-for-beginners-3f28).
- 📚 [**HTTP Tutorial – GeeksforGeeks**](https://www.geeksforgeeks.org/computer-networks/http-tutorial/): Covers HTTP methods, headers, status codes, and more [2](https://www.geeksforgeeks.org/computer-networks/http-tutorial/).

---

## ✅ **Step 2: Explore Database Basics**

- 🎥 [**Database Tutorial for Beginners** – YouTube (Lucid Software)](https://www.youtube.com/watch?v=wR0jg0eQsZA): A short, visual explanation using real-world examples [3](https://www.youtube.com/watch?v=wR0jg0eQsZA).
- 🎥 [**SQL Full Course for Beginners** – freeCodeCamp](https://www.youtube.com/watch?v=HXV3zeQKqGY): A comprehensive 4-hour course covering relational databases and SQL [4](https://www.youtube.com/watch?v=HXV3zeQKqGY).
- 🌐 [**SQL Tutorial – W3Schools**](https://www.w3schools.com/sql/): Interactive lessons and exercises to practice SQL queries [5](https://www.w3schools.com/sql/).

---

## ✅ **Step 3: Study API Concepts**

- 🎓 [**API Guide for Absolute Beginners – Free Udemy Course**](https://www.udemy.com/course/api-guide-for-absolute-beginners-free-course/): Learn what APIs are, how they work, and how to use them [6](https://www.udemy.com/course/api-guide-for-absolute-beginners-free-course/).
- 📖 [**How to Use REST APIs – freeCodeCamp**](https://www.freecodecamp.org/news/how-to-use-rest-api/): A beginner-friendly guide to REST APIs with practical examples [7](https://www.freecodecamp.org/news/how-to-use-rest-api/).

## ✅ **Step 4: Investigate Browser Storage Options**

- 📘 [**Browser Storage Options Explained – TechBuzzOnline**](https://techbuzzonline.com/web-developmentbrowser-storage-options-beginners-guide/): Covers cookies, localStorage, sessionStorage, IndexedDB, and Cache Storage [8](https://techbuzzonline.com/web-developmentbrowser-storage-options-beginners-guide/).
- 📖 [**A Guide to Web Storage – DEV Community**](https://dev.to/austinwdigital/a-guide-to-web-storage-localstorage-sessions-cookies-more-1fbm): Includes code examples and best practices [9](https://dev.to/austinwdigital/a-guide-to-web-storage-localstorage-sessions-cookies-more-1fbm).
