## Epic Overview

Still confused by Epics, User Stories, or Story Points ?
You’re not the problem ;) the jargon is.

Here’s Agile in plain English:
- 🏠 Epic = the house you’re building the big vision.
- 🛋️ Feature = the rooms major components.
- 🛠️ User Story = the actual work “Install lighting,” “Paint walls.”
- 📊 Story Points = effort, not hours. Bigger = more work.

Simple rule: if you can explain Agile like building a house you truly get it.
