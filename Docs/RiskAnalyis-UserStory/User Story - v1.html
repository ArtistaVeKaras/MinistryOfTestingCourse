User story 1

Task manager

Title

Add a task to the task list

Story

As a user,
I want to enter a task in a text field and add it to a list,
So that I can keep track of my tasks in a simple and organized way.

Acceptance criteria

  • The user can enter a task in the input field
  • Clicking the "Add Task" button adds the task to the list
  • The task is displayed below the input field
  • The input field is cleared after the task is added
  • The list should show multiple tasks as they are added
