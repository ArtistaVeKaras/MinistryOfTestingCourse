User story 2

Task manager with priority selection

Title

Add a task with a selectable priority

Story

As a user,
I want to assign a priority (high, medium, or low) when adding a task,
So that I can visually categorize and manage my tasks based on their urgency.

Acceptance criteria

  • The user can enter a task in the input field
  • The user can select a priority (high, medium, low) from a dropdown
  • Clicking the "Add task" button adds the task to the list with the selected priority
  • The task appears with a background color corresponding to its priority:
High priority Medium priority Low priority
  • The input field and priority selection reset after adding a task
